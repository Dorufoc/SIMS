快查指令使用指南

什么是快查指令？

快查指令提供常用的SQL语句模板和关键词速查，帮助您快速编写和执行SQL查询。无需记忆复杂的语法，通过参考示例即可轻松完成数据查询。

如何使用？

  1. 在下方SQL编辑器中输入或粘贴SQL语句
  2. 点击"执行"按钮运行查询
  3. 查看执行结果和生成的SQL
  4. 可点击"导出"将结果保存为CSV文件

常用示例

SELECT * FROM student_info 查询所有学生
SELECT * FROM student_info WHERE major = '计算机' 条件查询
SELECT * FROM student_info WHERE student_name LIKE '%张%' 模糊查询
SELECT * FROM student_info ORDER BY age DESC 排序查询
SELECT major, COUNT(*) FROM student_info GROUP BY major 统计查询

注意事项

  • 仅支持SELECT查询语句，确保数据安全
  • 危险操作（如DROP、DELETE）需要管理员权限
  • 查询结果可点击表头进行排序
  • 使用关键词速查区快速了解SQL语法

SQL语句执行器

暂无匹配数据
查询语句
基础查询 查询所有学生信息 
SELECT s.student_no, s.student_name, s.gender, s.age, c.class_name, c.grade, m.major_name, d.dept_name FROM students s LEFT JOIN classes c ON s.class_id = c.class_id LEFT JOIN majors m ON c.major_id = m.major_id LEFT JOIN departments d ON m.dept_id = d.dept_id;
指定列查询 查询指定字段 
SELECT s.student_no, s.student_name, m.major_name FROM students s LEFT JOIN classes c ON s.class_id = c.class_id LEFT JOIN majors m ON c.major_id = m.major_id;
条件查询 WHERE子句过滤 
SELECT s.student_no, s.student_name, s.gender, s.age, c.class_name, c.grade, m.major_name, d.dept_name FROM students s LEFT JOIN classes c ON s.class_id = c.class_id LEFT JOIN majors m ON c.major_id = m.major_id LEFT JOIN departments d ON m.dept_id = d.dept_id WHERE m.major_name = '计算机';
排序查询 ORDER BY排序 
SELECT s.student_no, s.student_name, s.gender, s.age, c.class_name, c.grade, m.major_name, d.dept_name FROM students s LEFT JOIN classes c ON s.class_id = c.class_id LEFT JOIN majors m ON c.major_id = m.major_id LEFT JOIN departments d ON m.dept_id = d.dept_id ORDER BY s.age DESC;
聚合统计
COUNT计数 统计总记录数 
SELECT COUNT(*) AS 总数 FROM students;
分组统计 GROUP BY分组 
SELECT m.major_name AS 专业, COUNT(*) AS 人数 FROM students s LEFT JOIN classes c ON s.class_id = c.class_id LEFT JOIN majors m ON c.major_id = m.major_id GROUP BY m.major_name;
聚合函数 AVG/MAX/MIN 
SELECT AVG(s.age) AS 平均年龄, MAX(s.age) AS 最大年龄, MIN(s.age) AS 最小年龄 FROM students s;

表结构操作
CREATE TABLE 创建数据表 
CREATE TABLE students (
  student_id INT AUTO_INCREMENT PRIMARY KEY,
  student_no VARCHAR(20) NOT NULL UNIQUE,
  student_name VARCHAR(30) NOT NULL,
  gender VARCHAR(6),
  age TINYINT,
  class_id INT,
  create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY (class_id) REFERENCES classes(class_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8;
ALTER TABLE 修改表结构 
ALTER TABLE students ADD COLUMN phone VARCHAR(20) DEFAULT NULL COMMENT '联系电话';
